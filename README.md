QUESTION ONE: Identify examples of IaaS, PaaS, and SaaS from your daily interactions.
Answer:

# IaaS (Infrastructure as a Service):
You control servers, OS, and deployment

Examples of IaaS
-Amazon Web Services EC2 (virtual servers)
-Microsoft Azure Virtual Machines
-Google Cloud Platform Compute Engine

Real-life usage:
Hosting your Express.js backend on a VPS
Running MongoDB on a cloud server


# PaaS (Platform as a Service)
You focus on code, provider handles infrastructure

Examples of PaaS
-AWS Elastic Beanstalk
-Google App Engine
-Heroku

Real-life usage:
Deploying your Node.js app without managing servers
Auto-scaling apps

# SaaS (Software as a Service)
Ready-to-use apps via browser

Examples of SaaS
-Gmail
-Google Docs
-Dropbox

Real-life usage:
Sending emails
Writing documents online
Storing files


## QUESTION 2: Research and present the offerings of at least two major cloud providers

# Amazon Web Services (AWS)

Amazon Web Services

Key Services:-
EC2: Virtual servers (IaaS)
S3: Storage
RDS: Managed databases
Elastic Beanstalk: App deployment (PaaS)
Lambda: Serverless

Strengths:
Most widely used
Huge service ecosystem
Highly scalable


# Google Cloud Platform (GCP)

Google Cloud Platform
Key Services:

Compute Engine: IaaS
App Engine: PaaS
Cloud Storage: Storage
BigQuery: Data analytics

Strengths:
Strong in AI & data
Simple UI
Good free tier
